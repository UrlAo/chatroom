# chatroom

Python Socket 聊天室（计算机网络课程设计）

## 项目结构

- `server.py` - 命令行版服务器
- `gui_server.py` - GUI 版服务器
- `client.py` - 命令行版客户端
- `gui_client.py` - GUI 版客户端
- `start_system.py` - 系统启动器

## 功能特点

- 支持多用户同时在线聊天
- 用户名注册机制
- 消息广播功能
- 用户上线/下线通知
- 服务器管理功能（查看在线用户、踢出用户、发送系统广播）
- GUI 图形界面，提供更好的用户体验
- 消息持久显示，用户可看到自己发送的消息

## 运行方式

1. 使用启动器（推荐）：`python start_system.py`
2. 或者分别启动：
   - 启动服务器：`python gui_server.py`
   - 启动客户端：`python gui_client.py`
3. 在客户端连接到服务器后即可开始聊天

## GUI 服务器功能

- 图形化服务器管理界面
- 实时显示服务器日志
- 在线用户列表显示
- 支持踢出指定用户
- 支持发送系统广播消息
- 服务器启动/停止控制
